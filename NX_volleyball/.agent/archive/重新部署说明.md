
执行编译：
```bash
cd ~/NX_volleyball/ros2_ws

rm -rf build/ install/ log/

# 重新编译（会自动安装模型文件到 install 目录）
source /opt/ros/humble/setup.bash

# 加载环境
source install/setup.bash
```

### 验证模型文件安装


## 3️⃣ 运行新节点

### ⭐ 最简启动（推荐）

**终端1 - 主追踪节点**
```bash
sudo -E bash -c "source /opt/ros/humble/setup.bash && \
source ~/NX_volleyball/ros2_ws/install/setup.bash && \
ros2 run volleyball_stereo_driver volleyball_tracker_node"
```

**终端2 - 可视化节点**
```bash
source /opt/ros/humble/setup.bash
source ~/NX_volleyball/ros2_ws/install/setup.bash
ros2 run volleyball_stereo_driver volleyball_visualizer_node
```

> 💡 **说明**：节点会自动使用内置默认参数，无需指定配置文件

---

### 自定义配置（可选）

如需修改参数，可指定配置文件：

```bash
# 主节点
sudo -E bash -c "source /opt/ros/humble/setup.bash && \
source ~/NX_volleyball/ros2_ws/install/setup.bash && \
ros2 run volleyball_stereo_driver volleyball_tracker_node \
--ros-args --params-file ~/NX_volleyball/ros2_ws/install/volleyball_stereo_driver/share/volleyball_stereo_driver/config/tracker_params.yaml"

# 可视化节点
ros2 run volleyball_stereo_driver volleyball_visualizer_node \
--ros-args --params-file ~/NX_volleyball/ros2_ws/install/volleyball_stereo_driver/share/volleyball_stereo_driver/config/visualizer_params.yaml
```

## 4️⃣ 查看运行状态

### 主节点输出
- 初始化信息（PWM、相机、YOLO、立体匹配器）
- 每100帧输出一次统计信息（帧率、延迟）
- 状态变化提示（GLOBAL_SEARCH ↔ ROI_TRACKING）

### 可视化节点窗口
- **主窗口**：显示检测框、3D位置、速度、轨迹
- **双目窗口**：左右相机实时画面（可选）
- **快捷键**：
  - `Q` / `ESC` - 退出
  - `S` - 截图
  - `T` - 切换轨迹显示
  - `C` - 清除轨迹历史

### ⚠️ 警告：未找到模型文件

**症状**:
```
[WARN] ⚠️  未找到模型文件: model/yolo11n.engine
```

**原因**: 模型文件路径查找失败

**解决方案**:

#### 方法1：重新编译（推荐）
编译会自动将模型文件从 `src/` 复制到 `install/` 目录：
```bash
cd ~/NX_volleyball/ros2_ws
rm -rf build/ install/ log/
colcon build --packages-select volleyball_stereo_driver
source install/setup.bash
```

#### 方法2：验证模型文件
优化后的代码会在多个位置搜索：
```bash
# 检查源目录
ls -lh ~/NX_volleyball/ros2_ws/src/volleyball_stereo_driver/model/yolo11n.engine

# 检查安装目录
ls -lh ~/NX_volleyball/ros2_ws/install/volleyball_stereo_driver/share/volleyball_stereo_driver/model/yolo11n.engine
```

#### 方法3：转换模型（如果缺失）
如果模型文件不存在，需要转换：
```bash
cd ~/NX_volleyball
python3 scripts/convert_yolo_to_tensorrt.py \
  --model ros2_ws/src/volleyball_stereo_driver/model/best.pt \
  --output ros2_ws/src/volleyball_stereo_driver/model/yolo11n.engine \
  --imgsz 640 \
  --fp16
```

### ⚠️ 立体匹配不准确

当前使用的是占位符标定参数。要获得准确的3D位置：

1. **运行标定程序**:
```bash
cd ~/NX_volleyball/calibration
python3 capture_chessboard.py
```

2. **替换标定文件**:
将生成的 `stereo_calib.yaml` 复制到：
```bash
cp stereo_calib.yaml ~/NX_volleyball/ros2_ws/src/volleyball_stereo_driver/calibration/
```

3. **重新编译**:
```bash
cd ~/NX_volleyball/ros2_ws
colcon build --packages-select volleyball_stereo_driver
```
