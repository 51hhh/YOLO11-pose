# 问题速查：网络代理配置

> 环境: Jetson Orin NX (Ubuntu 22.04) · 代理服务器 192.168.31.30:10000

## 问题
NX 无法直接访问外网，pip/wget/curl/apt 均超时。

## 解决

### 终端代理
```bash
export http_proxy=http://192.168.31.30:10000
export https_proxy=http://192.168.31.30:10000
export no_proxy=localhost,127.0.0.1,192.168.31.*
```

### pip 代理
```bash
pip install xxx --proxy http://192.168.31.30:10000
# 或永久写入
mkdir -p ~/.config/pip
cat > ~/.config/pip/pip.conf << 'EOF'
[global]
proxy = http://192.168.31.30:10000
EOF
```

### apt 代理
```bash
sudo tee /etc/apt/apt.conf.d/proxy.conf << 'EOF'
Acquire::http::Proxy "http://192.168.31.30:10000";
Acquire::https::Proxy "http://192.168.31.30:10000";
EOF
```

### wget
```bash
echo 'use_proxy=yes
http_proxy=http://192.168.31.30:10000
https_proxy=http://192.168.31.30:10000' >> ~/.wgetrc
```

## 验证
```bash
curl -x http://192.168.31.30:10000 https://pypi.org/simple/ -o /dev/null -w "%{http_code}\n"
# 应返回 200
```

## 注意
- 代理端口范围 10000-10010（经扫描验证 10000 可用）
- JetPack wheel 下载需代理：`wget -e use_proxy=yes ...`
