# 01 项目总览

## 系统简介

排球双目3D追踪系统，基于 YOLO11-Pose 姿态估计 + 双目立体视觉，实现排球的实时检测、追踪与三维定位。部署于 Jetson Orin NX Super 16GB，实测 **98+ FPS**（100Hz PWM硬件触发）。

## 技术栈

| 层次 | 技术 | 说明 |
|------|------|------|
| 检测模型 | YOLO26n (自定义) | 1类+5关键点 Pose, 640×640, Pre-NMS |
| 推理引擎 | TensorRT 10.3 | GPU INT8 2.89ms/345qps |
| 图像管线 | VPI 3.2.4 | Remap校正(CUDA), 自定义Bayer→BGR CUDA kernel |
| 立体匹配 | CUDA ROI SAD | 检测区域5×5网格多点匹配, <0.1ms |
| 深度融合 | Kalman 滤波 | 单目(0-4m) + 双目(4-15m) 混合 |
| 相机 | 海康 MV-CA016-10UC ×2 | USB3, BayerRG8 1440×1080, 硬件触发 |
| 同步 | libgpiod PWM | 100Hz 精确同步双目 |
| 平台 | JetPack 6.2 (L4T R36.4.7) | CUDA 12.6, Ubuntu 22.04 |

## 核心模块一览

| 模块 | 源文件 | 功能 | 硬件 |
|------|--------|------|------|
| Pipeline | pipeline.h/cpp | 四阶段流水线调度, 三缓冲, 异步Grab | CPU+GPU |
| FrameSlot | frame_slot.h | 三帧环形缓冲区, VPI/CUDA资源池 | GPU Memory |
| HikvisionCamera | hikvision_camera.h/cpp | MVS SDK双目采集, 零拷贝 | USB3 |
| TrtDetector | trt_detector.h/cpp | TensorRT YOLO推理, DLA/GPU | DLA/GPU |
| DetectPreprocess | detect_preprocess.cu | Bayer→BGR CUDA kernel, Letterbox | GPU |
| VpiRectifier | vpi_rectifier.h/cpp | VPI Remap畸变校正 | CUDA/VIC |
| ROIStereoMatcher | roi_stereo_matcher.h/cpp | ROI多点SAD块匹配 | GPU |
| HybridDepth | hybrid_depth.h/cpp | 单目+双目Kalman滤波融合 | CPU |
| Coordinate3D | coordinate_3d.h/cpp | 视差→3D三角测距 | GPU |
| PWMTrigger | pwm_trigger.h | libgpiod精确100Hz触发 | GPIO |

## 数据流

```
PWM 100Hz ──→ 海康双目 (BayerRG8)
                   │
           ┌───────┴───────┐
           ↓               ↓
      左相机原始       右相机原始
           │               │
      CUDA Bayer→BGR   CUDA Bayer→BGR
           │               │
      VPI Remap校正    VPI Remap校正
           │               │
      BGR送检测        灰度送匹配
           │               │
    TRT YOLO检测     ←── 检测框ROI
           │               │
      目标框+关键点    ROI SAD匹配
           │               │
           └───────┬───────┘
                   ↓
          三角测距 → 3D坐标
                   ↓
          Kalman深度融合
                   ↓
           输出: [x, y, z, vx, vy, vz]
```
