# 05 部署指南

## 硬件环境

| 项目 | 规格 |
|------|------|
| SoC | Jetson Orin NX 16GB Super |
| GPU | 8 SM, Compute Capability 8.7 |
| CUDA | 12.6.68 |
| TensorRT | 10.3.0.30 |
| VPI | 3.2.4 |
| JetPack | 6.2 (L4T R36.4.7) |
| OS | Ubuntu 22.04 |
| 电源模式 | MAXN_SUPER (mode 0) |

## 环境搭建

### 1. 系统优化

```bash
# 切换到最高性能模式
sudo nvpmodel -m 0
sudo jetson_clocks

# 验证
nvpmodel -q    # 应显示 MAXN_SUPER
```

### 2. 依赖安装

```bash
# 系统依赖
sudo apt install -y \
  libopencv-dev \
  libyaml-cpp-dev \
  libgpiod-dev \
  cmake \
  build-essential

# 海康 MVS SDK (手动安装)
# 下载 MVS Linux aarch64 版本, 解压后:
sudo dpkg -i MVS-*.deb
```

### 3. 构建

```bash
cd NX_volleyball/stereo_3d_pipeline
mkdir -p build && cd build
cmake .. -DCMAKE_BUILD_TYPE=Release
make -j4
```

### 4. TensorRT 引擎构建

```bash
# GPU FP16 (推荐)
cd scripts
bash build_engine.sh ../model/yolo26.onnx

# 验证引擎
trtexec --loadEngine=../model/yolo26_fp16.engine --warmUp=1000 --iterations=100
```

### 5. 运行

```bash
# 生产模式 (ROI + 100Hz触发)
./build/stereo_pipeline -c config/pipeline_yolo26_mixed.yaml

# 调试模式 (带可视化)
./build/stereo_pipeline -c config/pipeline_yolo26_mixed.yaml --show

# 自由运行 (无PWM)
./build/stereo_pipeline -c config/pipeline_roi_freerun.yaml
```

## 性能调优工具

### tegrastats (实时监控)

```bash
tegrastats --interval 1000
# 输出: CPU/GPU 频率、温度、内存、功耗
```

### jtop (交互式监控)

```bash
jtop
# 注意: jtop 4.3.2 的 JetPack 6.2 映射缺失
# 修复: 在 /usr/lib/python3/dist-packages/jtop/core/jetson_variables.py 中
# NVIDIA_JETPACK 字典添加 "36.4.7": "6.2"
```

### nsys (GPU Profiling)

```bash
nsys profile --trace=cuda,nvtx --output=profile \
  ./build/stereo_pipeline -c config/pipeline_yolo26_mixed.yaml

nsys stats profile.nsys-rep
```

### trtexec (引擎基准测试)

```bash
trtexec --loadEngine=model.engine \
  --warmUp=1000 \
  --duration=10 \
  --useCudaGraph \
  --separateProfilerRun
```

## CMake 条件编译

| 开关 | 说明 | 默认 |
|------|------|------|
| HIK_CAMERA_ENABLED | 海康 SDK 支持 | 自动检测 |
| NO_GPIOD | 禁用 libgpiod (软件PWM) | OFF |
| HAS_ONNXRUNTIME | ONNX Runtime 深度模型 | 自动检测 |
| HAS_XIMGPROC | OpenCV contrib WLS 滤波 | 自动检测 |
