# 07 训练流程

## 概述

使用 Ultralytics YOLO11-Pose 框架训练排球检测+关键点模型。  
任务: 1类 (volleyball) + 5关键点 (球心Center + 4个轮廓点)。

## 1. 数据准备

### 数据量建议

| 集合 | 图像数 | 说明 |
|------|--------|------|
| 训练集 | 800-1000 | 多场景多光照 |
| 验证集 | 200 | 不同于训练集 |
| 测试集 | 200 | 最终评估 |

### 标注规范

- 格式: YOLO Pose (txt)
- 关键点: 5个, 顺序: Center(球心), Top, Right, Bottom, Left
- **Center 必须精确** — 几何拟合圆的中心点
- 使用 Label Studio 或 CVAT 标注工具

### 数据集配置 (dataset.yaml)

```yaml
path: ./data
train: train/images
val: val/images
test: test/images
nc: 1
names: ['volleyball']
kpt_shape: [5, 3]  # 5个关键点, 每个(x,y,visible)
```

## 2. 模型训练

### 硬件要求

- GPU: RTX 3060+ (6GB VRAM 最低)
- RAM: 16GB+
- 推荐: RTX 4090 (大batch训练)

### 训练命令

```bash
cd volleyball_tracking/train
pip install -r requirements.txt

python train.py \
  --model yolo11n-pose.pt \
  --data ../data/dataset.yaml \
  --epochs 300 \
  --imgsz 640 \
  --batch 16
```

### 训练配置 (config.yaml)

```yaml
model: yolo11n-pose.pt
data: ../data/dataset.yaml
epochs: 300
imgsz: 640
batch: 16
optimizer: AdamW
lr0: 0.001
weight_decay: 0.05
```

### 训练技巧

- 使用预训练权重 (`yolo11n-pose.pt`) 加速收敛
- 数据增强: Mosaic + MixUp + 随机翻转
- 关键点可见性 flag 必须正确设置
- 验证集 mAP@0.5 目标 > 0.9

## 3. TensorRT 导出

### 方法一: Ultralytics 直接导出

```python
from ultralytics import YOLO

model = YOLO('best.pt')
model.export(format='engine', imgsz=640, half=True)
```

### 方法二: ONNX → TensorRT

```bash
# 1. 导出 ONNX
python -c "
from ultralytics import YOLO
model = YOLO('best.pt')
model.export(format='onnx', imgsz=640, simplify=True)
"

# 2. 构建 TRT 引擎 (在 NX 上执行)
trtexec --onnx=best.onnx --saveEngine=best_fp16.engine --fp16
```

### 注意事项

- ONNX 导出时使用 `simplify=True` 减少冗余算子
- TRT 引擎必须在目标设备 (Jetson NX) 上构建, 不跨平台
- JetPack 6.2 使用 TRT 10.3, 与桌面 TRT 版本不兼容
- 验证输出格式: Pre-NMS (bbox+conf+kpts) vs Post-NMS (含TopK)
- **推荐 Pre-NMS 格式**: 避免 DLA 不支持 TopK 的问题

### 配置更新

修改 `config/pipeline_yolo26_mixed.yaml`:
```yaml
detector:
  engine_path: "model/best_fp16.engine"
  input_width: 640
  input_height: 640
  conf_threshold: 0.25
  nms_threshold: 0.45
  num_classes: 1
  num_keypoints: 5
```
