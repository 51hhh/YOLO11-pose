# 单目 & 双目测距原理

## 1. 单目测距 (Monocular Depth)

### 原理

利用**已知物体尺寸**和**透视投影**关系，从 2D 图像推算距离。

```
Z_mono = f × D_real / w_px
```

| 符号 | 含义 | 值 |
|------|------|-----|
| f | 焦距 (像素) | 1671.2 px (校正后) |
| D_real | 排球直径 | 0.23 m |
| w_px | BBox 宽度 (像素) | 检测输出 |
| Z_mono | 距离 (m) | 计算得 |

### 数值表

```
Z_mono = 1671.2 × 0.23 / w_px = 384.4 / w_px

距离(m)  BBox宽(px)  单像素误差
  1.0      384        ±0.003m (±3mm)
  2.0      192        ±0.010m (±1cm)
  3.0      128        ±0.023m (±2.3cm)
  5.0       77        ±0.065m (±6.5cm)
 10.0       38        ±0.26m  (±26cm)
```

### 优缺点

- **优点**：仅需单目，计算量极小 (1次除法)
- **缺点**：精度受 BBox 量化限制；远距离误差增大；依赖准确的物体尺寸

### 测量目标

**球心到相机的距离**。

原因：公式中 D_real 是球的直径（中心到中心的跨度），而 w_px 是球在图像上的投影宽度。
投影关系 `w_px = f × D_real / Z` 中的 Z 是球心到像平面的距离，因此反算出的也是球心距离。

如需球表面距离：`Z_surface = Z_center - radius = Z_center - 0.115m`

## 2. 双目测距 (Stereo Depth)

### 原理

左右相机同时拍摄，同一目标在两张图中的水平位移（**视差 d**）与距离成反比。

```
Z_stereo = f × B / d
```

| 符号 | 含义 | 值 |
|------|------|-----|
| f | 焦距 (像素) | 1671.2 px |
| B | 基线 (m) | 0.3287 m |
| d | 视差 (像素) | SAD 匹配得 |

### 匹配策略

本项目使用 **ROI SAD 匹配**：
1. 在左目图中用 YOLO 检测得到 BBox → 确定 ROI 区域
2. 在 ROI 内取 5×5 网格采样点
3. 对每个点，在右目对应行搜索最小 SAD (Sum of Absolute Differences)
4. 取加权中值视差 → 计算深度

### 测量目标

**球心到相机的距离**。

原因：SAD 匹配在球的中心区域取样，得到的视差对应的是球中心位置的深度。
球体表面最靠近相机的点比球心近一个半径，但匹配区域覆盖整个球面投影，
中值滤波后结果趋近球心深度。

## 3. 混合策略 (Hybrid Fusion)

```
距离 < 3m  →  纯单目 (双目视差超搜索范围)
3-5m       →  加权融合: Z = α×Z_mono + (1-α)×Z_stereo
               α = (5 - Z) / 2  线性过渡
距离 > 5m  →  纯双目 (单目精度不够)
```

融合后经 **Kalman 滤波** 时域平滑。

---

## 4. 误差来源

| 因素 | 单目影响 | 双目影响 |
|------|---------|---------|
| BBox 边缘跳动 ±1px | 近距小、远距大 | 无 |
| 球变形/遮挡 | BBox 变小→距离高估 | 匹配质量下降 |
| 标定误差 | 焦距误差→系统偏移 | 基线+焦距双重 |
| 光照变化 | 间接(影响检测) | 直接(影响SAD) |
| 抖动/运动模糊 | BBox 模糊放大 | 匹配失败 |

---

## 5. 精度验证方法

1. 将排球放在**已知距离**处（用卷尺测量）
2. 运行管线采集 N 帧 (建议 ≥50 帧)
3. 记录每帧的 Z_mono, Z_stereo, Z_fused
4. 计算与真值的偏差 (bias) 和标准差 (std)
5. 多距离重复 → 绘制误差曲线
