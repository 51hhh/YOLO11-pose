# 代码地图

最后核对: 2026-07-03

本页按文件定位当前实现。查行为先看对应源码，再回到 Wiki 更新稳定结论。

## 顶层入口

| 文件 | 职责 |
|---|---|
| `src/main.cpp` | 解析命令行和 YAML，初始化 Pipeline、记录器、ROS2、显示和调试模式 |
| `src/main_config_loaders.cpp` | 顶层 YAML 配置加载和各配置域分发 |
| `src/main_dual_yolo_config.*` | `detector.dual_yolo` 深度候选、fallback 和特征匹配配置映射 |
| `src/main_recording_config_loaders.cpp` | 预测、轨迹录制、baseline clip 和 realtime debug dump 配置加载 |
| `src/main_runtime_overrides.*` | CLI 覆盖 baseline clip / realtime debug dump 配置，以及 baseline 模式下的 pipeline 降级 |
| `src/stereo_depth_viewer.cpp` | 双目深度 viewer、headless 算法对比和诊断 |
| `CMakeLists.txt` | 目标、依赖、条件编译和安装规则 |

## detect

| 文件 | 职责 |
|---|---|
| `trt_detector.h/.cpp` | TensorRT detector、slot buffer、enqueue/collect、DFL 输出适配 |
| `detect_preprocess.cu` | Bayer/Gray/BGR 预处理 CUDA kernel |

## rectify

| 文件 | 职责 |
|---|---|
| `vpi_rectifier.h/.cpp` | VPI/CUDA/PVA 校正、WarpMap、灰度/BGR remap |

## pipeline

| 文件 | 职责 |
|---|---|
| `pipeline.h` | `Pipeline` 主类、配置、FrameSlot/async ROI 私有接口 |
| `pipeline.cpp` | 初始化、Stage0/1/2/3、dual YOLO ROI 候选生成、legacy `z_stereo` 兼容选择、debug feature、回调 |
| `pipeline_loops.cpp` | `pipelineLoop()`、`pipelineLoopROI()` 和性能报告；只放主调度循环 |
| `pipeline_async_roi.cpp` | async ROI Stage2 资源、snapshot、worker、完成队列和过期丢弃 |
| `frame_slot.h` | `FrameSlot`、`FrameMetadata`、VPI/CUDA 图像资源、检测结果和生命周期清理 |
| `object3d_types.h` | `Object3D` 在线 3D 观测、候选深度、质量、pair gate 和水印字段 |
| `detection_types.h` | 2D 检测结构 |
| `sync.h` | CUDA/VPI stream 管理辅助 |

关键函数:

| 函数 | 作用 |
|---|---|
| `pipelineLoopROI()` | ROI 模式三段重叠调度 |
| `stage0_grab_and_rectify()` | 抓帧、TNR、debayer、remap 和 gray/BGR 准备 |
| `stage1_detect()` | TensorRT YOLO enqueue |
| `stage2_roi_match_fuse()` | 检测帧 ROI 候选、深度和融合 |
| `submitAsyncRoiStage2()` | 检测帧 collect 后提交 async ROI snapshot 和 worker task |
| `asyncRoiWorkerLoop()` | 在 worker 自有 buffer 上执行完整 ROI Stage2，过期则只释放不写回 |
| `stage2_roi_fuse_tracker()` | tracker 填充帧 ROI 融合 |
| `debugFeatureMatchesOnce()` | 单帧特征匹配调试输出 |

## stereo

| 文件 | 职责 |
|---|---|
| `dual_yolo_depth_gpu.h/.cpp/.cu` | 左右 YOLO pair 的 GPU ROI 深度候选和验证 |
| `depth_candidate_builder.h/.cpp` | 将各路 `z_*` 和质量字段组装成有序 `DepthCandidateObservation`，并生成 legacy P0/fallback 兼容选择 |
| `depth_candidate_contract.cpp` | 深度候选方法/source 映射、可用性判断和 legacy allow-list 选择 |
| `depth_match_contract.h/.cpp` | 左右 YOLO pair gate 合同、reject reason 统计和 pair 排序 |
| `roi_stereo_matcher.h/.cpp`, `roi_stereo_match.cu` | legacy ROI 立体匹配 |
| `roi_circle_match.cu` | ROI 圆/边缘相关 CUDA 实现 |
| `roi_feature_match_cpu.h/.cpp` | CPU sparse/OpenCV 特征匹配和 debug 可视化；OpenCV extractor/descriptor helper 在 `roi_feature_match_opencv_helpers.h` |
| `roi_geometry_cpu.h/.cpp` | CPU ROI 边缘质心、径向中心和成对边缘中心 |
| `roi_geometry_circle_cpu.cpp` | CPU ROI 圆拟合、bbox proxy、圆心 detection 转换和 fallback 极线圆搜索 |
| `roi_patch_match_cpu.h/.cpp` | CPU ZNCC/Census patch 匹配 |
| `neural_feature_config.h` | 神经特征 backend 和门限配置 |
| `neural_feature_matcher.h/.cpp` | TensorRT fused 神经特征 ROI 匹配 |
| `vpi_stereo.h/.cpp` | VPI 全帧/半分辨率 stereo |
| `onnx_stereo.h/.cpp` | viewer 可选 ONNX stereo 模型 |

## fusion

| 文件 | 职责 |

| `depth_extract.cu` | CUDA 深度提取辅助 |
