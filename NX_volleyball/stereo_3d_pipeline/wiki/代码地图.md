| `src/main_runtime_overrides.*` | CLI 覆盖 baseline clip / realtime debug dump 配置，以及 baseline 模式下的 pipeline 降级 |
| `src/main_display_helpers.h` | 主显示 helper 兼容聚合头 |
| `src/main_display_types.h` | 显示点击测距状态和 display job 数据结构 |
| `src/main_display_capture.h` | 显示用 color remap、显示尺寸计算和 VPI/CUDA frame 抓取 |
| `src/main_display_render.h` | 检测框、深度、预测、点击测距和 HUD 绘制 |
| `src/main_realtime_debug_dump.cpp` | realtime debug dump 采样、队列、VPI gray copy 和生命周期 |
| `src/main_realtime_debug_dump_writer.cpp` | realtime debug dump zoom 图、summary JSON、PNG 写出和 writer loop |
| `src/stereo_depth_viewer.cpp` | 双目深度 viewer、headless 算法对比和诊断 |
| `src/stereo_depth_viewer_vpi.h` | viewer VPI 资源、remap、上传下载和 VPI CUDA SGM 计算 |
| `src/stereo_depth_viewer_depth_utils.h` | viewer 视差转深度、灰度显示、双边/中值/去斑点后处理 |

## pipeline

| 文件 | 职责 |
|---|---|

| `sync.h` | CUDA/VPI stream 管理辅助 |

关键函数:

| 函数 | 作用 |
|---|---|
| `pipelineLoopROI()` | ROI 模式三段重叠调度 |
| `stage0_grab_and_rectify()` | 抓帧、TNR、debayer、remap 和 gray/BGR 准备 |
| `stage1_detect()` | TensorRT YOLO enqueue |
| `stage2_roi_match_fuse()` | 检测帧 ROI 候选、深度和融合 |
| `submitAsyncRoiStage2()` | 检测帧 collect 后提交 async ROI snapshot 和 worker task |
| `asyncRoiWorkerLoop()` | 在 worker 自有 buffer 上执行完整 ROI Stage2，过期则只释放不写回 |
| `stage2_roi_fuse_tracker()` | tracker 填充帧 ROI 融合 |
| `debugFeatureMatchesOnce()` | 单帧特征匹配调试输出 |
