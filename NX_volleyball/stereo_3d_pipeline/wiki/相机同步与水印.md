# 相机同步与水印

最后核对: 2026-07-02

相机封装在 `src/capture/hikvision_camera.*`。当前系统使用海康双目外触发，水印字段用于判断左右同步和触发稳定性。

## 相机配置

`CameraConfig` 关键字段:

| 字段 | 说明 |
|---|---|
| `serial_left`, `serial_right` | 推荐绑定方式，避免 USB 枚举顺序变化 |
| `exposure_us`, `gain_db` | 固定曝光和增益 |
| `auto_exposure`, `auto_gain` | 自动曝光/增益及边界 |
| `gamma_enable`, `gamma_value` | 相机 gamma |
| `use_trigger` | 外触发模式 |
| `trigger_source`, `trigger_activation` | 触发线和边沿 |
| `trigger_frequency_hz` | 用于帧计数追帧和超时判断 |
| `image_node_num` | SDK FIFO 深度，吸收 USB 到达抖动 |
| `embedded_info_clear_rows` | 清除 FrameSpecInfo 写入图像首部的行数 |
| `width`, `height` | 采集尺寸 |

当前主配置使用:

```yaml
serial_left: "00D39342665"
serial_right: "00219471413"
use_trigger: true
trigger_source: "Line0"
trigger_activation: "RisingEdge"
image_node_num: 3
embedded_info_clear_rows: 2
width: 1440
height: 1080
```

## 抓帧结果

`GrabResult` 字段:

| 字段 | 来源 | 用途 |
|---|---|---|
| `success` | SDK 抓帧结果 | 判断本帧是否可用 |
| `timestamp_us` | SDK 设备时间戳原值，当前海康 USB 实测为 ns | 诊断左右 timestamp delta |
| `host_timestamp` | 主机生成时间戳 | 诊断 USB/调度 jitter |
| `frame_number` | SDK 帧号 | 辅助同步检查 |
| `frame_counter` | 水印帧计数 | 主同步键 |
| `trigger_index` | 水印外触发计数 | 触发稳定性诊断 |

## FrameMetadata

`makeFrameMetadata()` 从 `FrameSlot` 生成回调和记录用 metadata:

| 字段 | 计算方式 |
|---|---|
| `frame_counter_delta` | `left_frame_counter - right_frame_counter` |
| `frame_number_delta` | `left_frame_number - right_frame_number` |
| `timestamp_delta_us` | `(left_timestamp_us - right_timestamp_us) / 1000` |
| `grab_failed` | 当前 slot 采集或校正失败 |
| `is_detect_frame` | 当前帧是否提交 YOLO |

100fps 实机测试至少应确认:

- `frame_counter_delta` 稳定为 0。
- `trigger_index` 连续递增，无周期性跳变。
- `timestamp_delta_us` 稳定，无明显漂移。
- `Stage0_WaitGrab` outlier 不频繁。

## 水印探针

构建出 `hik_frame_metadata_probe` 后，可先独立测试相机和 PWM:

```bash
./hik_frame_metadata_probe \
  --left-sn 00D39342665 \
  --right-sn 00219471413 \
  --frames 500 \
  --timeout-ms 100 \
  --exposure-us 9867 \
  --gain-db 11.9906 \
  --trigger-chip gpiochip2 \
  --trigger-line 7 \
  --pwm-hz 100
```

如果外部 PWM 已经由其他进程提供:

```bash
./hik_frame_metadata_probe \
  --left-sn 00D39342665 \
  --right-sn 00219471413 \
  --frames 500 \
  --no-pwm
```

## 问题定位

| 现象 | 优先检查 |
|---|---|
| `frame_counter_delta` 非 0 | 左右相机触发线、序列号绑定、SDK FIFO、是否有旧进程占用相机 |
| `trigger_index` 跳变 | PWM 稳定性、GPIO 线、相机触发模式 |
| `timestamp_delta_us` 周期性波动 | USB 带宽、主机调度、`image_node_num` |
| YOLO y 残差系统性偏移 | 标定/校正质量，必要时评估 `feature_y_slope` 和重新标定 |
| 画面首行异常影响检测 | `embedded_info_clear_rows` 是否匹配水印行数 |
