# 命令行参考

最后核对: 2026-07-02

本页只列当前工具实际支持的主要参数。新增、删除或改名参数后必须同步更新。

## stereo_pipeline

文件: `src/main.cpp`

```bash
./stereo_pipeline [--config <path>] [--visualize] \
  [--debug-feature-matches] [--debug-feature-matches-dir <dir>] \
  [--record-baseline-clip] [--baseline-out <dir>] \
  [--baseline-duration <sec>] [--baseline-frames <n>] \
  [--baseline-clips <n>] [--baseline-gap <sec>] \
  [--baseline-format <png|pgm>] [--baseline-image-mode <gray|bgr|both>] \
  [--baseline-start-immediately]
```

| 参数 | 说明 |
|---|---|
| `--config`, `-c` | Pipeline YAML 配置 |
| `--visualize`, `--display`, `-v` | 显示检测和距离 overlay 窗口 |
| `--debug-feature-matches` | 抓一对校正图并导出 ROI 特征匹配图片 |
| `--debug-feature-matches-dir DIR` | 指定特征匹配调试输出目录，同时启用 debug |
| `--record-baseline-clip` | 检测到球后录制固定长度左右图序列和 CSV |
| `--baseline-out DIR` | baseline clip 输出根目录 |
| `--baseline-duration SEC` | 按触发频率换算录制帧数 |
| `--baseline-frames N` | 精确录制 N 帧，优先于 duration |
| `--baseline-clips N` | 连续录制 N 段 clip |
| `--baseline-gap SEC` | 两段 clip 之间跳过的时间 |
| `--baseline-format png|pgm` | 无损图像格式 |
| `--baseline-image-mode gray|bgr|both` | 选择保存灰度或彩色；正式录制通常用 `gray` 或 `bgr`, `both` 仅用于调试兼容并会额外输出 `left_bgr/`、`right_bgr/` |
| `--baseline-start-immediately` | 不等待检测框，启动后立即按帧录制 |
| `--help`, `-h` | 显示帮助 |

兼容项:

- `--visualizels` 会被当作 `--visualize`，用于避免常见拼写错误导致静默关闭显示。

## capture_chessboard

文件: `src/calibration/capture_chessboard.cpp`

```bash
./capture_chessboard -o calibration_images -g 17.0 \
  --serial-left 00D39342665 \
  --serial-right 00219471413
```

| 参数 | 说明 |
|---|---|
| `-o DIR` | 输出目录 |
| `-e US` | 曝光时间，单位微秒 |
| `-g DB` | 增益 dB |
| `--free-run` | 自由运行，不使用硬触发 |
| `--no-pwm` | 禁用 PWM 输出 |
| `--headless` | 无 GUI |
| `-n COUNT` | 自动保存 COUNT 对并退出 |
| `--width W`, `--height H` | 相机分辨率 |
| `--left-index N`, `--right-index N` | 按索引绑定相机 |
| `--serial-left S`, `--serial-right S` | 按序列号绑定相机 |
| `--image-node-num N` | SDK FIFO 深度 |

## stereo_calibrate

文件: `src/calibration/stereo_calibrate.cpp`

```bash
./stereo_calibrate -s 26.0 --board-w 5 --board-h 8
```

| 参数 | 说明 |
|---|---|
| `-s`, `--square-size MM` | 棋盘格边长，单位 mm，必填 |
| `-d`, `--images-dir DIR` | 图像目录 |
| `-o`, `--output FILE` | 输出标定 YAML |
| `--board-w N`, `--board-h N` | 内角点数量 |
| `--no-vis` | 跳过校正预览 |
| `--gpu-preprocess` | CUDA 预处理 Bayer/BGR 到灰度 |
| `--optimize-intrinsics` | stereoCalibrate 联合优化内参 |
| `--sb` | classic 失败后启用 SB fallback |
| `--exhaustive` | 启用慢速 SB exhaustive fallback |
| `--jobs N` | 并行检测 worker 数 |

## stereo_depth_viewer

文件: `src/stereo_depth_viewer.cpp`

```bash
./stereo_depth_viewer -c calibration/stereo_calib.yaml [options]
```

| 参数 | 说明 |
|---|---|
| `-c`, `--calibration` | 标定 YAML |
| `--config PATH` | pipeline YAML 配置 |
| `--max-disp N` | 最大视差 |
| `--win-size N` | 匹配窗口 |
| `--free-run` | 自由运行，无外触发，不启动 PWM |
| `--pwm-freq HZ` | PWM 触发频率 |
| `--no-pwm` | 禁用 PWM |
| `--diagnose` | 诊断模式 |
| `--swap-lr` | 左右相机互换 |
| `--headless` | 无窗口，自动遍历模式 |
| `--frames N` | headless 每种算法运行帧数 |
| `--crestereo PATH` | CREStereo ONNX 模型 |
| `--hitnet PATH` | HITNet ONNX 模型 |

### export_xfeat_extractor_onnx.py

```bash
python3 scripts/export_xfeat_extractor_onnx.py \
  --xfeat-repo /path/to/accelerated_features \
  --out models/neural/xfeat_extractor_224.onnx \
  --roi-size 224
```

| 参数 | 说明 |
|---|---|
| `--xfeat-repo` | `verlab/accelerated_features` checkout，必填 |
| `--weights` | `xfeat.pt` 路径；为空时使用 `<xfeat-repo>/weights/xfeat.pt` |
| `--out` | 输出 ONNX 路径，必填 |
| `--roi-size` | 固定 ROI 输入尺寸，默认 224，必须能被 32 整除 |
| `--opset` | ONNX opset，默认 17 |

`scripts/nx_build_xfeat_extractor_engine.sh` 使用环境变量配置:

| 环境变量 | 说明 |
|---|---|
| `ROOT_DIR` | `stereo_3d_pipeline` 根目录 |
| `XFEAT_REPO` | `verlab/accelerated_features` checkout |
| `OUT_DIR` | ONNX/engine 输出目录 |
| `ROI_SIZE` | 固定 ROI 尺寸，需能被 32 整除 |
| `OPSET` | ONNX opset |
| `TRT_FP16` | `1` 时构建 FP16 engine |
