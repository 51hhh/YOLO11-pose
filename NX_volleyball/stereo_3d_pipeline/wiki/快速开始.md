# 快速开始

## 在 NX 上构建

```bash
cd ~/NX_volleyball/stereo_3d_pipeline
cmake -S . -B build_standalone -DCMAKE_BUILD_TYPE=Release -DCUDA_ARCH=87 \
  -DCMAKE_DISABLE_FIND_PACKAGE_ament_cmake=ON \
  -DCMAKE_CUDA_COMPILER=/usr/local/cuda-12.6/bin/nvcc \
  -DCUDAToolkit_ROOT=/usr/local/cuda-12.6
cmake --build build_standalone -j$(nproc)
```

构建目标和条件依赖见 [构建目标与依赖](构建目标与依赖.md)。

## 运行实时管线

```bash
cd ~/NX_volleyball/stereo_3d_pipeline/build_standalone
./stereo_pipeline --config ../config/pipeline_dual_yolo_roi.yaml
```

带显示:

```bash
./stereo_pipeline --config ../config/pipeline_dual_yolo_roi.yaml --visualize
```

运行配置字段见 [配置参考](配置参考.md)，实时调度细节见 [实时调度细节](实时调度细节.md)。

## 录制一段基准片段

基准片段用于后续离线对比不同特征点提取、匹配和深度求解算法。

```bash
./stereo_pipeline \
  --config ../config/pipeline_dual_yolo_roi.yaml \
  --record-baseline-clip \
  --baseline-out baseline_clips \
  --baseline-duration 3.0 \
  --baseline-format png
```

输出结构:

```text
baseline_clips/clip_YYYYMMDD_HHMMSS/
├── left/000000.png
├── right/000000.png
├── frames.csv
└── metadata.yaml
```

## 导出单帧特征匹配可视化

```bash
./stereo_pipeline \
  --config ../config/pipeline_dual_yolo_roi.yaml \
  --debug-feature-matches \
  --debug-feature-matches-dir debug_feature_matches
```

输出会包含左右校正灰度图、检测框图、每种方法的匹配图、contact sheet 和 summary。

连续帧评估流程和离线脚本见 [录制与离线评估](录制与离线评估.md) 与 [工具脚本索引](工具脚本索引.md)。

## 标定采集

```bash
./capture_chessboard \
  -o calibration_images \
  -g 17.0 \
  --serial-left 00D39342665 \
  --serial-right 00219471413
```

求解:

```bash
./stereo_calibrate -s 26.0 --board-w 5 --board-h 8
```

本机 Python 求解:

```bash
scp -r nvidia@10.42.0.148:~/NX_volleyball/stereo_3d_pipeline/build_standalone/calibration_images ./NX_volleyball/calibration
cd ./NX_volleyball/calibration
python3 stereo_calibration.py -s 26.0
```

完整标定参数和质量要求见 [标定工具细节](标定工具细节.md)，相机同步诊断见 [相机同步与水印](相机同步与水印.md)。
